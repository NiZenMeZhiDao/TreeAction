# R2 全场规则约束与机器人约束整理

> 来源：`第二十五届全国大学生机器人大赛ROBOCON_u201C武林探秘_u201D竞技赛规则V5.pdf`、`第二十五届全国大学生机器人大赛ROBOCON武林探秘图册V3-1.pdf`，以及当前 R2 设计方案中已明确的机器人能力边界。
> 本文只整理比赛规则和我方机器人约束。

## 1. 总体比赛规则

- 比赛为红蓝两队同时进行，每队两台机器人：R1 和 R2。
- 比赛时长为 4 分钟。
- R1 可以是手动或自动机器人。
- R2 必须是自动机器人；除启动外，比赛开始后不能手动控制。
- 比赛场地为 `12m x 12m` 正方形，四周有内高 `100mm`、厚 `50mm` 的木质围栏。
- 场地由北向南分为三个区域：武馆、梅林、对抗区。
- 红蓝半场之间有高 `100mm`、厚 `50mm` 的纵向木质隔板。
- R1/R2 短暂伸入对方半场上方空间允许，但伸入后与对方机器人接触会被强制重试。
- 场地外围 `1500mm` 宽区域为参赛队员和裁判活动区，队员只能在本队半场外围活动和操作。

## 2. R2 活动区域约束

R2 只能在本队以下区域运行：

- 武馆。
- 梅林中的 R2 入口区。
- 梅林中的树林。
- 梅林中的 R2 出口区。
- 对抗区。

R2 不允许：

- 进入对方半场。
- 在梅林中进入 R1 通道执行任务。
- 在梅林中收集 R1 KFS。
- 接触或移动假 KFS。
- 在武馆中接触长杆。
- 比赛中由队员手动控制。

## 3. 场地和道具规则

武馆：

- 比赛开始前，每队长杆架上有 4 根长杆。
- 端头架由两队共用，摆放 6 个端头，三种端头各 2 个。
- 武馆是启动、重试和组装兵器的区域。

梅林：

- 梅林用于收集武术秘籍 KFS。
- 梅林包含树林和 R1 通道。
- 树林由高低不同方块组成，只允许 R2 运行。
- 树林方块顶面有白色胶带标出的正方形标记，外廓边长 `355mm`。
- R1 通道只允许 R1 运行，但与 R2 入口区、出口区共用的两段通道除外。

KFS：

- KFS 是边长 `350mm` 的立方体纸板箱。
- 每队梅林内有 3 个 R1 KFS、4 个 R2 KFS、1 个假 KFS。
- R1 KFS 是 R1 的收集对象。
- R2 KFS 是 R2 的收集对象。
- 假 KFS 是禁止机器人接触的物品。
- 设置时间内，本队 KFS 位置由对手放置。

对抗区：

- 对抗区包含坡道、九宫格、重试区、已用兵器区。
- R1 和 R2 必须通过坡道进入和离开对抗区。
- 九宫格为 `3 x 3` 架子，用于放置 KFS。
- 只有空格才能放置 KFS。

## 4. 设置阶段约束

- 每队在比赛前有 1 分钟设置时间。
- 机器人和控制装置在比赛开始前必须完全纳入启动区及其上方空间。
- 对手负责把本队的 8 个 KFS 放入本队梅林中指定位置。
- KFS 必须放在树林方块中央的正方形边界内，且白线框四边均露出。
- 假 KFS 不得放在梅林入口处的 1、2、3 号方块上。
- 每个方块最多只能放一个 KFS，不得堆叠。
- KFS 侧面图案或文字应直立，顶面朝向没有要求。
- 若对方未能在规定时间完成本队 KFS 放置，对方会被视为违反公平竞争原则并取消比赛资格。

## 5. 武馆阶段规则

武馆阶段目标是组装兵器。

R2 相关规则：

- 两台机器人同时启动。
- R2 从共用端头架取端头。
- R2 抓取端头的位置由参赛队自定。
- R2 每次只能接触、拿起和移动 1 个端头。
- R2 抓取端头时不得用自己的机构影响对方抓取其它端头，否则会被取消比赛资格。
- 组装时 R1 应抓住长杆，R2 应抓住端头。
- 组装过程中，R2 不得接触 R1 抓住的长杆。
- 组装过程中，R1 和 R2 任何部位不得直接肢体接触。
- R2 在抓取另一端头前，必须先完成当前端头与长杆的组装。
- 已完成组装的兵器只能由 R1 携带，或存放在长杆架上。

离开武馆：

- R1 必须携带一件或多件组装好的兵器离开武馆前往梅林。
- 只有在 R1 完全离开武馆后，R2 才能离开武馆。
- 如果比赛开始后 2 分钟内仍未组装好一件兵器，且对方 R1/R2 均已离开武馆进入梅林，该队两台机器人可以按规则离开武馆进入梅林完成收集任务。

掉落物处理：

- 掉落到比赛场地外的端头或长杆不能再用。
- 只有 R2 能捡起掉落在本队武馆内的端头。
- 只有 R1 能捡起掉落在本队武馆内的长杆或组装好的兵器。
- 若机器人把长杆或端头掉落到对方武馆内，相关机器人强制重试。

## 6. 梅林阶段规则

梅林阶段目标是收集 KFS，为对抗区得分做准备。

R1 规则简述：

- R1 只能沿 R1 通道运动。
- R1 只应收集 R1 KFS。
- R1 拿取 R1 KFS 的数量不限。
- R1 绕行通道次数不限。
- 兵器在梅林中没有任务用途，不能用于完成梅林任务。

R2 规则：

- R2 只应收集 R2 KFS。
- R2 必须通过 R2 入口进入树林。
- R2 可以从当前位置拿取相邻方块上的 R2 KFS。
- R2 不得拿取与当前位置不相邻方块上的 R2 KFS。
- “相邻”指在场地俯视图上与 R2 当前位置有公共边的方块；斜向不算相邻。
- 如果 1、2、3 号方块上有 R2 KFS，R2 必须从 R2 入口处收集第一个 KFS。
- R2 离开树林前，必须收集并携带至少一个 R2 KFS。
- R2 必须经过指定的 10、11 或 12 号方块之一离开树林。
- R2 KFS 掉落在树林外不得再用。
- R2 可以捡起掉落在树林内的 R2 KFS。
- R2 即使能到达树林外的 R2 KFS，也不能捡起。

梅林禁止项：

- R1/R2 不得以任何方式接触或移动假 KFS。
- R1 不得在梅林中接触 R2 KFS。
- R2 不得在梅林中接触 R1 KFS。
- R2 不得完全进入已有 KFS 的树林方块。
- R2 不得违反相邻方块捡拾规则。

## 7. 对抗区阶段规则

进入对抗区：

- R1 第一次进入对抗区时，应携带至少一件组装好的兵器和/或 1 个 R1 KFS。
- R2 进入对抗区前，必须携带至少 1 个 R2 KFS。
- R1 和 R2 必须通过坡道进入和离开对抗区。

九宫格放置：

- R1 每次只能把一个 KFS 放到九宫格底层。
- R2 每次只能把一个 KFS 放到九宫格中层。
- 由 R1 举起的 R2 每次只能把一个 KFS 放到九宫格顶层。
- R1 举起 R2 的动作只能在对抗区进行。
- 顶层放置时，R2 不得与地面接触。
- 无论 R1、R2 或被 R1 举起的 R2，都只能把 KFS 放入九宫格空格。
- R2 放本队 KFS 时，每次只能放一个，不得同时把 KFS 放到中层和顶层。
- 放置在空格中的 KFS 位置和是否与格子边线平行，没有特别要求。

攻防：

- R1 可以且只能用持有的一件兵器移除对方九宫格中的 KFS。
- 每件组装好的兵器只能用一次。
- 兵器与任意 KFS 接触并脱离后即视为已用。
- 已用兵器在使用另一件兵器前必须放入已用兵器区。
- R2 不负责使用兵器移除对方 KFS。
- R1/R2/R1 举起的 R2 可用自身部件或本队 KFS 防止对方移除本队 KFS。
- 每次只能防守一个格子。
- R1 不得用兵器防守本队 KFS。

## 8. 结束和获胜规则

比赛立即结束的情况：

- 某队在九宫格的任一垂直列或对角线上放置了 3 个 KFS，且其中至少 1 个是 R2 KFS，获得“武术大师”称号。
- 4 分钟比赛时间到。
- 某队被取消比赛资格。

获胜顺序：

- 获得“武术大师”的队伍获胜。
- 若没有“武术大师”，总分高者获胜。
- 若总分相同，先比较九宫格得分。
- 若仍相同，比较对抗区总分。
- 若仍相同，比较武馆内总分。
- 若仍持平，由裁判确定胜队。

## 9. 得分规则

比赛结束后按两队各自半场上 KFS 和兵器的实际情况核查得分。比赛结束前已被破坏的得分状态不再记分。

| 项目 | 得分条件 | 分值 |
| --- | --- | --- |
| 组装兵器 | 本队 R1 持有或遗留在本队半场上的组装且未解体兵器 | 20 / 件 |
| 对抗区地面或机器人上 | 本队 R1 KFS | 20 / 个 |
| 对抗区地面或机器人上 | 本队 R2 KFS | 30 / 个 |
| 九宫格底层 | 本队 R1 KFS，且与机器人无接触 | 40 / 个 |
| 九宫格底层 | 本队 R2 KFS，且与机器人无接触 | 50 / 个 |
| 九宫格中层 | 本队 R1 KFS，且与机器人无接触 | 60 / 个 |
| 九宫格中层 | 本队 R2 KFS，且与机器人无接触 | 70 / 个 |
| 九宫格顶层 | 本队 R1 KFS，且与机器人无接触 | 90 / 个 |
| 九宫格顶层 | 本队 R2 KFS，且与机器人无接触 | 100 / 个 |

对 R2 的策略含义：

- R2 KFS 在中层和顶层价值更高。
- 顶层 R2 KFS 单个价值最高，但必须依赖 R1 举起 R2。
- 中层放置是 R2 自身必须可靠完成的核心得分动作。
- 若追求“武术大师”，九宫格成线的 3 个 KFS 中至少需要 1 个 R2 KFS。

## 10. 犯规和强制重试

以下行为会导致强制性重试：

- R1/R2 伸入对方比赛场地上方并与对方机器人接触。
- R1 接触武馆内端头。
- R2 接触武馆内长杆。
- R1 第一次出武馆时未携带组装好的兵器。
- R2 第一次出武馆时先于 R1 出武馆。
- R1 在梅林中接触 R2 KFS。
- R2 在梅林中接触 R1 KFS。
- R1 或 R2 接触和移动假 KFS。
- R2 完全进入有 KFS 的树林方块。
- R2 违反入口相邻区捡拾规则。
- R1 再次使用已用兵器，或使用新兵器前未丢弃已用兵器。
- 参赛队员未经裁判允许与机器人接触。
- 防守队对超过一个九宫格格子做出阻挡动作，或用兵器阻挡。

取消比赛资格风险：

- 有悖公平竞争精神的行为。
- 未听从裁判指令或警告。
- 机器人严重损坏比赛场地、设施、设备或对方机器人，从而影响比赛进行。
- R2 抓端头时用机构影响对方抓取其它端头。
- 将本队 KFS 放入非空九宫格，按具体条款可能导致强制重试或取消资格。

## 11. 重试规则

- 机器人故障或策略需要时，参赛队可以申请重试。
- 裁判可以对犯规机器人进行强制重试。
- 被强制重试的机器人回到指定重试区后，必须停留 15 秒才能在裁判允许下重新启动。
- 一场比赛中重试次数不限。
- 重试期间比赛时钟不停。
- 未重试的本队机器人和对方机器人比赛活动不停。
- 重试期间，队员只可调整机器人持有的 KFS、长杆、端头或组装兵器。
- 队员不得调整场地上未被机器人持有的物品。
- 若因接触假 KFS 强制重试，裁判或志愿者会把被移动的假 KFS 归位。

重试位置：

- 武馆或梅林中重试：R1 回武馆 R1 重试区，R2 回武馆 R2 重试区。
- 武馆或梅林中 R1/R2 同时重试：分别回武馆各自重试区。
- 对抗区中参赛队申请重试：可选回对抗区重试区或武馆对应重试区。
- 对抗区中强制重试：犯规机器人只能回对抗区重试区；若两台都强制重试，R1 先、R2 后。

## 12. 官方机器人硬约束

数量和控制：

- 每队只能有两台机器人：R1 和 R2。
- R1 可手动或自动。
- R2 必须能够独立运行，是自动机器人。

尺寸：

| 机器人 | 比赛开始前尺寸上限 | 比赛中完全伸展尺寸上限 |
| --- | --- | --- |
| R1 | 宽 1000 x 长 1000 x 高 1000 mm | 宽 1000 x 长 1800 x 高 1300 mm |
| R2 | 宽 800 x 长 800 x 高 800 mm | 宽 800 x 长 1300 x 高 1300 mm |

重量：

- R1 和 R2 总重量，包括电池、控制器、电缆，不得超过 `50kg`。
- 机器人尺寸和重量是最大值，没有允许误差。

通信：

- 射频通信只能使用 Wi-Fi、Zigbee、蓝牙，用于控制器和机器人之间通信。
- 组委会不管理 Wi-Fi、Zigbee、蓝牙环境。
- 比赛中 R1 和 R2 不得用无线传输相互通信。
- 规则不禁止其它非无线通信方式。

能源：

- 只能使用电池、压缩空气和/或弹性力作为能源。
- 所有电池标称电压不得超过 `24V`。
- 电源电路任意实测最高电压不得超过 `42V`。
- 使用压缩空气时，气压不得超过 `600kPa`，且只能作为气动元件气源。

禁用：

- 铅酸电池。
- 粘合密封电池。
- 易燃易爆和高温能源产品。
- 高噪音和强气流的涵道风机。
- 可能损坏场地或妨碍比赛的物品。
- 无人机、飞行装置、移动投射物、任何空中移动方式。

安全：

- 所有机器人必须安装红色硬件急停按钮。
- 机器人失控危及安全时，队员可按急停，并以不影响其它机器人的方式将其移出场外；除重试外不得返回。
- 机器人设计不得威胁人员、场地、周围环境或其它机器人。
- 队员比赛和试运行时必须穿胶鞋、戴头盔。
- 场地、设施尺寸重量若无特别说明误差为 `±5%`，但机器人尺寸和重量无误差余量。

## 13. 我们 R2 的当前能力约束

以下是我方 R2 当前设计和策略需要遵守的内部约束，应作为机械、电控、规划和调试共同检查项。

自主性：

- R2 必须按自动机器人设计，启动后不依赖手动控制。
- R2 与 R1 的配合不能依赖无线通信。
- 若需要 R1/R2 交互，应优先采用机械接触、视觉标志、局部传感器、车体状态显示等非无线方式。

尺寸重量：

- R2 启动前必须完全小于 `800 x 800 x 800 mm`。
- R2 完全展开后必须小于 `800 x 1300 x 1300 mm`。
- R1/R2 总重上限为 `50kg`，R2 设计重量需要给 R1 留足余量。
- 举升上层放置时，R2 的展开高度和 R1 举升结构叠加后仍要满足规则动作边界和安全要求。

武馆能力：

- R2 只能操作端头，不能碰长杆。
- R2 一次只能拿一个端头。
- R2 的端头抓取机构不能覆盖或干扰多个端头的投影区域。
- 组装兵器时，R2 只负责端头侧，不能接触 R1 的长杆侧。
- 组装过程中 R2 与 R1 不得发生直接肢体接触。

梅林移动能力：

- R2 在梅林只服务 R2 KFS，不处理 R1 KFS 和假 KFS。
- R2 必须从 R2 入口进树林，从 10、11、12 号方块之一出树林。
- R2 的路径不得规划为完全进入已有 KFS 的方块。
- R2 只能抓取与当前方块有公共边的相邻方块上的 R2 KFS。
- 斜向抓取、隔格抓取、跨方块抓取均不应作为有效能力。
- R2 需要具备在高低方块间稳定通行的能力。
- R2 在上下台阶、抓取、携带 KFS 时都不能碰假 KFS。
- 兵器不能作为 R2 在梅林中完成任务的工具。

KFS 携带和路径策略：

- R2 离开树林前至少要携带一个 R2 KFS。
- R2 进入对抗区前至少要携带一个 R2 KFS。
- 当前保守策略不应规划一次携带全部 4 个 R2 KFS。
- 当前策略优先保证可稳定拿到至少 2 个 R2 KFS，再根据机械容量和时间决定是否拿第 3 个。
- 若机械结构确认可以稳定携带更多 KFS，策略约束需要同步更新。

对抗区能力：

- R2 必须能携带 KFS 通过坡道进入对抗区。
- R2 自身只能放置九宫格中层 KFS。
- R2 顶层放置必须依赖 R1 举起，且放置时 R2 不得与地面接触。
- R2 每次只能放一个 KFS。
- R2 不能把 KFS 放入非空格。
- R2 放置后必须与 KFS 脱离接触，否则不能稳定记分。
- R2 不使用兵器移除对方 KFS。

安全和可靠性：

- R2 必须有清晰可触达的红色急停按钮。
- 所有展开机构、抓取机构、举升配合结构不得形成伤人、夹人或损坏场地的风险。
- R2 在被 R1 举起时必须有稳定支撑/定位设计，避免滑落或倾覆。
- R2 掉落、卡死、定位明显失效或误碰假 KFS 风险高时，应优先选择停机或申请重试，而不是继续执行。

## 14. 需要继续确认的队内约束

这些不是规则原文，而是需要队内尽快定下来的设计指标：

- R2 单机目标重量，以及 R1/R2 之间的重量分配。
- R2 实际 KFS 最大携带数量。
- R2 是否携带 R1 KFS 参与得分，以及是否需要把 R1 KFS 传给 R1。
- R1 举起 R2 的接触点、限位方式、触发确认方式。
- R2 顶层放置时是否存在姿态变化导致超尺寸或不稳定风险。
- R2 通过坡道时的最大坡度、最小离地间隙和防倾覆裕量。
- R2 在梅林方块上的最小安全支撑面积和允许悬空量。
- R2 对假 KFS、R1 KFS、R2 KFS 的识别可靠性要求。
- 端头抓取机构是否会影响相邻端头或对方抓取空间。
- 非无线 R1/R2 通信方式和失效处理。
